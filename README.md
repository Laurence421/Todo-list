## ✨ Features

*   **Core Task Management**: Lightweight functionality focusing on the two most essential user actions: adding new tasks and deleting completed ones.
*   **Fully Responsive Layout**: Built with a mobile-first design that adjusts fluidly across mobile, tablet, and desktop screens.
*   **Clean & Minimalist UI**: An uncluttered interface designed to keep users focused on their immediate tasks.

## 🛠️ Tech Stack

*   **HTML5**: Semantic structure for accessibility and task inputs.
*   **CSS3 / [Tailwind if used]**: Responsive layout styling (using Flexbox/Grid).
*   **JavaScript (Vanilla)**: DOM manipulation to dynamically create, render, and remove task elements.
